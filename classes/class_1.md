# Menu Class

## Attributes
- entree_names (array)
- market_price (float)
- has_fries (boolean)
- current_special (string)

## Methods
- update_special (rewrites `current_special`)
- set_price (updates `market_price`)
- remove_entree (removes an entree from `entree_names` array)
- add_fries (sets `has_fries` to true)
